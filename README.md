# 🚀 Smart To-Do List App with LocalStorage

A dynamic, fully responsive task management application built using **Vanilla JavaScript, HTML5, and CSS3**. This project focuses on implementing efficient data persistence, interactive DOM manipulation, and strict input validation.

---

## ✨ Features

- **Multi-Input Task Creation:** Users can add tasks by specifying the *Task Name*, *Duration*, and *Priority Level*.
- **Data Persistence:** Integrated browser `LocalStorage` using `JSON` parsing to save tasks permanently across page reloads.
- **Dynamic DOM Rendering:** Instantly updates the UI upon adding or deleting tasks while ensuring no element duplication.
- **Robust Input Validation:** Prevents empty or invalid entries with real-time error feedback for the user.
- **Global State Reset:** A dedicated "Delete All" feature that clears both the browser storage and the active UI in one click.

---

## 🛠️ Tech Stack

- **Structure:** HTML5
- **Styling:** CSS3 (Modern, high-contrast theme)
- **Logic & State:** Vanilla avaScript (ES6+)
